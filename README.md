## üìñ Usage

### Setting Your Hostname
1. On first visit, a modal will prompt you to enter a hostname
2. Enter a valid hostname (letters, numbers, hyphens)
3. Click "Set Hostname" or press Enter
4. Your hostname will appear in all command examples

### Navigating Modules
- Click **Login Management** to learn device security configuration
- Click **VLAN Configuration** to learn Layer 2 segmentation
- Use the navbar links to jump to specific sections

### Copying Commands
- Click the **Copy** button on any terminal window
- A checkmark and "Copied" text confirms success
- A toast notification appears at the bottom
- Paste directly into Packet Tracer!
